## Design Pattern List

### Behavioural Patterns

- Chain of Responsibility
- Command
- Interpreter
- Mediator
- Memento
- Null Object
- Observer
- State
- Strategy
- Template Method
- Visitor

### Creational Patterns

Creational patterns help you to create new objects from scratch in an easy way.

- Builder
- Factory
  - Abstract Factory
- Object Pool
- Prototype
- Singleton

### Structural Patterns

- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Private Class Data
- Proxy
